# dash-phpdoc

Generates a [PHP](http://php.net/) docset for [Dash](http://kapeli.com/dash), with multi-language support. 

Modified from https://github.com/toyjack/dash-phpdoc-ja .

## requirement

- php
- hh.exe, MSYS (Windows)
- chmlib (Mac, Linux)

## usage

modify configs in `generate.php` to fit your needs.

```shell
php generate.php
```
